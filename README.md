# DLC Unhider
Mod for PAYDAY 2 that forces game to not hide certain unowned DLC items like Scarface's room upgrade, Kingpin perk deck, anniversary items and other event stuff.

## Installation
1. Install [SuperBLT](https://superblt.znix.xyz/)
2. Download this repo as zip
3. Unpack zip to mods folder of PAYDAY 2
